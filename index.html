Algunos comandos para linux

pwd (Print working directory)

Imprime la dirección en la cual se encuentra actualmente el usuario

ls (List directory)

Imprime los ficcheros y directorios contenidos en tu ubicacion o directorio actual

cd (Change directory)

Nos permite moversnos entre directorios mediantes rutas relativas o absolutas.

touch

Crea archivos vacios, si existen actualiza la fecha y hora de modificación.Ejm de formatos(.rb,.css,.js)

echo

Imprime en la terminal de comandos el mensaje escrito , también tiene otras funciones extra.

file

Nos muestra las características de uno o más archivos

diff

Muestra las diferencias entre dos archivos, muy útil para comparar código

find

Busca un archivo o direcctorio que le especifiques, devuelve la dirección.

grep

Busca la palabra indicada dentro de un archivo y muestra las concidencias.
